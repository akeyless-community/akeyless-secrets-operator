# Contributing

External contributions are welcome via pull request.

- **Merge policy:** Only `@akeyless-community/cs-admin` team members can merge to `main`, after CI passes and at least one approving review.
- **Fork PRs:** External contributors can open PRs; a team member must approve and merge.

See the [Developer guide](docs/contributing/devguide.md) for local setup, and the [Contributing Process](docs/contributing/process.md) for the PR workflow.
